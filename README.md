# XYUISlider

### 初始化

```
self.umberSlider = [[BAUISlider alloc] initWithFrame:CGRectMake(50, 100, self.view.frame.size.width-100, 30)];
//显示方向
self.umberSlider.titleStyle = TopTitleStyle;
//是否显示百分比
self.umberSlider.isShowTitle = YES;
```
